Your website should contain the following pages:

1 Homepage (index.html)
This page should include:
Your full name
Your photograph
A welcome message
A navigation menu
A short biography
This is the first page visitors will see.

2 About Me Page
This page should contain:
Educational background
Career aspirations
Technical skills
Hobbies and interests
Keep it neat and professional.

3 Projects Page
You are expected to showcase at least three projects.
For each project include:
Project title
Description
Screenshot/image
Project link (if available)
If you do not have many projects yet, you can create simple sample projects for demonstration purposes.

4 Academic Planner Page
This is the interactive part of the project.
Users should be able to:
Add tasks
Mark tasks as completed
Delete tasks
Example:
Study HTML
Complete Lab Assessment
Attend Group Discussion
This section will require JavaScript.

5 Contact Page
Include:
Name field
Email field
Phone number field
Message field
  JavaScript Validation Requirements
The contact form must be validated.
Examples:
Empty Fields
If a user leaves a field empty:
"Please enter your name."
Email Validation
The email must be properly formatted.
Correct:
student@gmail.com
Incorrect:
studentgmail.com
Phone Validation
Phone numbers should contain only numbers.
Correct:
08012345678
