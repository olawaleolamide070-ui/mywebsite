PixelHub — Design and Visual Concepts

Web

Personal Portfolio Website

This portfolio website is a fully responsive, multi-page site built entirely with vanilla HTML, CSS, and JavaScript — no frameworks or libraries. It features a clean, modern design with smooth scroll animations, a mobile-responsive layout, and interactive elements like the academic task planner and contact form validation.


Key highlights include a dynamic navigation bar with scroll effects, Intersection Observer-based reveal animations, a fully functional task management system, and a validated contact form. The entire project demonstrates my ability to build polished, production-quality web experiences from scratch.
